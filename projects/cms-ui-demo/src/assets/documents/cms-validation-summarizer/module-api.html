API

Name Data type Type Description
group-id string Input Id of group the validation summarizer belongs to. This can be used for identifying whether to apply validation summarizer item template builder or not.
instance AbstractControl | NgControl | null Input Instance of the control that needs to be validated.
label string Input Label to be displayed in the validation message.
validation-template TemplateRef<any> | null Input Custom template for validation messages.
maximum-messages number Input
Maximum number of messages which can be displayed in validation summarizer control
By default, every validation message will be displayed
visibility-handler (ngControl: AbstractControl | NgControl) => boolean Input
To control how and when validation messages will be displayed.
Normally, validation message will be displayed when control is dirty & touched
