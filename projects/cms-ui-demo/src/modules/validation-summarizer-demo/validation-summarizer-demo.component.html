Basic validation summarizer
  • Validation summarizer can be placed under a specific control
  • It retrieves control errors and display them on the screen.
Summarizer with custom validator
  • This validation summarizer also supports custom validation rule.
  • Additional message mapping must be defined to be displayed onto screen.
  • For more information, please input value which is smaller than 10 into Age field.
Summarizer with visibility handler
  • Visibility handler can be used for deciding whether validation messages can be visible or not
  • If no handler is used, service handler will be used instead.
  • Component handler has higher priority than the service one
Template driven form
  • Validation summarizer also supports template driven form
